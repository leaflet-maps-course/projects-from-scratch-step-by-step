3.- Leaflet Curve, para crear una curva de transición entre dos rutas
Normalmente se usa en transiciones de vuelos o viajes
https://www.npmjs.com/package/leaflet-curve
npm i leaflet-curve
Ejemplo:
https://stackoverflow.com/a/53507336
Artículo medium explicando como hacer la curva en un punto intermedio:
https://ryancatalani.medium.com/creating-consistently-curved-lines-on-leaflet-b59bc03fa9dc

4.- Leaflet Heatmap
Para añadir los mapas de calor
https://www.npmjs.com/package/leaflet.heat

5.- Leaflet Ant Path
Para añadir línea personalizadas y en movimiento, como si estuvieran hormigas
npm install leaflet-ant-path --save
Soporte para polyline, polygon, curve, circle, rectangle
https://rubenspgcavalcante.github.io/leaflet-ant-path/documentation#including-installing
